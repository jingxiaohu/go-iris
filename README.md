# Golang iris restful后台权限系统

#### 介绍
golang iris casbin jwt vue实战，支持websocket。可前后端分离。

### 背景介绍
Iris的教程较少、零散、基础，且框架集合的完整实战案例极少(毕竟多数是用于工作，不是demo)，几乎没有。可以直接使用。<br />
源于开源，馈与社区。<br />
